## 概要：
- TradingView 上に描画する

## 目的：
- 毎回手作業で"トレンドライン・水平線"を引くことが面倒だったのでそれを自動化している。

## 構成　：
```
.
├── horizontal_line
│   ├── 15minHighLow.pine
│   ├── 1hHighLow.pine
│   ├── 1minHighLow.pine
│   ├── 4hHighLow.pine
│   ├── 5minHighLow.pine
│   ├── dayHighLow.pine
│   └── yearHighLow.pine
└── README.md
```

## 使い方：
- Copy and Paste で Pine エディタに貼り付けて実行<br>
※ TradingView に GitHub の連携 API が存在しない為。
